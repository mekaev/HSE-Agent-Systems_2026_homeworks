# Skill: Расчёт и интерпретация космических расстояний

## Требуемые инструменты
- get_planet
- get_star
- get_mission
- convert_units
- calculate_light_travel

## Когда активировать
Этот skill активируется, когда пользователь:
- Спрашивает о расстоянии между объектами
- Хочет понять масштабы космоса
- Спрашивает «как далеко», «сколько лететь», «как долго свет идёт»
- Интересуется единицами измерения в астрономии

## Протокол действий

### Шаг 1. Определи объекты
- Какие объекты? Планеты, звёзды, другое?
- Используй `get_planet`, `get_star`, `get_mission` для получения данных о расстояниях
- Для планет расстояние дано в AU (от Солнца)
- Для звёзд расстояние дано в световых годах (от Земли/Солнца)

### Шаг 2. Вычисли расстояние
Для расстояния между двумя планетами:
- Приближённое расстояние = |distance_au₁ - distance_au₂| AU
- Это минимальное расстояние (при максимальном сближении)
- Реальное расстояние зависит от позиций на орбитах

Для расстояния до звезды:
- Используй distance_ly из данных звезды

### Шаг 3. Конвертируй в разные единицы
Обязательно покажи расстояние в НЕСКОЛЬКИХ единицах:
1. **Астрономические единицы (AU)** — для масштабов Солнечной системы
2. **Световые годы (ly)** — для межзвёздных расстояний
3. **Километры (km)** — для интуитивного понимания
4. **Парсеки (parsec)** — для профессионального контекста

Используй `convert_units` для точной конвертации.

### Шаг 4. Рассчитай время полёта света
Используй `calculate_light_travel` для получения:
- Время в годах, днях, часах, минутах
- Аналогию для понимания масштаба

### Шаг 5. Создай «шкалу масштабов»
Помоги понять расстояние через аналогии:

```
🚀 РАССТОЯНИЕ: [Объект A] → [Объект B]
═══════════════════════════════════════

📏 Абсолютное расстояние:
   • X км
   • Y AU
   • Z световых лет

⏱️ Время полёта света: ...

🏃 Аналогии для понимания:
   • Если Солнечную систему уменьшить до футбольного поля,
     то [объект] был бы на расстоянии [аналогия]
   • На самолёте (900 км/ч) полёт занял бы [время]
   • На «Вояджере-1» (17 км/с) — [время]

📡 Для справки:
   • Расстояние Земля → Солнце = 1 AU = 149.6 млн км = 8 мин 19 сек (свет)
   • Расстояние до Проксимы Центавра = 4.24 св. года
   • Диаметр Млечного Пути ≈ 100 000 св. лет
```

### Шаг 6. Контекст путешествий
Если уместно, добавь:
- Сколько бы заняло путешествие на современных технологиях
- Скорость «Вояджера-1» ≈ 17 км/с ≈ 61 200 км/ч
- Скорость света ≈ 300 000 км/с

## Стиль ответа
- Начинай с точного ответа (число + единица)
- Затем раскрывай в разных единицах
- Обязательно давай аналогии — масштабы космоса неинтуитивны
- Используй эмодзи для структуры (📏, ⏱️, 🚀)
- Всегда показывай «шкалу» — от знакомого к неизвестному
