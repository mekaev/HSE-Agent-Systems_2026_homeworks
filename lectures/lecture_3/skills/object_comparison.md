# Skill: Систематическое сравнение космических объектов

## Требуемые инструменты
- get_planet
- get_star
- get_constellation
- list_planets
- list_stars
- search_objects
- convert_units
- calculate_light_travel

## Когда активировать
Этот skill активируется, когда пользователь хочет:
- Сравнить два или более космических объекта
- Найти самый большой / маленький / далёкий / яркий объект
- Понять различия между типами объектов
- Составить рейтинг или ранжирование

## Протокол действий

### Шаг 1. Определи объекты для сравнения
- Выясни, какие конкретно объекты сравниваются
- Если пользователь сказал «все планеты» или «звёзды Ориона» — получи полный список
- Используй `list_planets`, `list_stars`, `search_objects` для получения списков

### Шаг 2. Собери данные по КАЖДОМУ объекту
Для корректного сравнения нужны данные о каждом объекте:
- Используй `get_planet` / `get_star` / `get_constellation` для каждого
- НЕ ПРОПУСКАЙ объекты — сравнение должно быть полным
- Запоминай ключевые параметры для сравнения

### Шаг 3. Выбери критерии сравнения
В зависимости от типов объектов выбери релевантные критерии:

**Планеты:**
| Критерий | Единица |
|----------|---------|
| Масса | масс Земли |
| Радиус | км |
| Расстояние от Солнца | AU |
| Период обращения | дней |
| Количество спутников | шт |
| Температура поверхности | °C |

**Звёзды:**
| Критерий | Единица |
|----------|---------|
| Расстояние | св. лет |
| Видимая величина | mag |
| Температура | K |
| Масса | масс Солнца |

### Шаг 4. Выполни вычисления
Если нужны производные величины:
- Используй `convert_units` для приведения к общим единицам
- Используй `calculate_light_travel` для понимания масштабов расстояний
- Считай соотношения (во сколько раз один больше другого)

### Шаг 5. Представь результат в виде таблицы
Всегда оформляй сравнение как структурированную таблицу:

```
📊 СРАВНЕНИЕ: [Объект A] vs [Объект B]
═══════════════════════════════════════
| Параметр          | Объект A    | Объект B    | Соотношение  |
|-------------------|-------------|-------------|--------------|
| Масса             | ...         | ...         | A в X раз >  |
| Радиус            | ...         | ...         | B в Y раз >  |
| ...               | ...         | ...         | ...          |

🏆 Итог: [краткий вывод — кто "победил" и по каким параметрам]
```

### Шаг 6. Добавь инсайты
- Укажи самые удивительные различия
- Приведи аналогии для понимания масштабов
  (например: «Юпитер в 11 раз больше Земли по диаметру — если Земля это вишня, то Юпитер — баскетбольный мяч»)
- Упомяни fun_fact для каждого объекта

## Стиль ответа
- Структурированный и аналитический
- Таблицы обязательны
- Числа с единицами измерения
- Соотношения для наглядности (в X раз больше/меньше)
- Завершай кратким выводом
