# Skills — Навыки космического ассистента

Набор skills, определяющих **как** агент должен действовать в типичных сценариях работы с астрономическими данными через MCP-инструменты.

## Список навыков

| Навык | Файл | Краткое описание |
|-------|------|------------------|
| Расчёт расстояний | [distance_calculation.md](distance_calculation.md) | Вычисление и интерпретация космических расстояний между объектами с конвертацией единиц (AU, ly, km, parsec) и расчётом времени полёта света |
| Сравнение объектов | [object_comparison.md](object_comparison.md) | Систематическое сравнение планет, звёзд и других космических объектов по ключевым параметрам с представлением результатов в виде таблиц и рейтингов |
| Планирование наблюдений | [observation_planning.md](observation_planning.md) | Составление плана астрономических наблюдений: выбор объектов, оценка видимости, рекомендации по оборудованию и времени наблюдения |

## Как работают skills

Каждый skill содержит:
1. **Триггеры** — при каких запросах пользователя активируется навык
2. **Протокол действий** — пошаговая инструкция, какие MCP-инструменты вызывать и в каком порядке
3. **Формат вывода** — шаблон структурированного ответа пользователю
