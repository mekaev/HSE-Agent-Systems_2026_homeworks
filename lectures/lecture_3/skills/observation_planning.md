# Skill: Планирование астрономических наблюдений

## Требуемые инструменты
- list_planets
- get_planet
- list_stars
- get_star
- get_constellation
- search_objects

## Когда активировать
Этот skill активируется, когда пользователь хочет:
- Узнать, что можно наблюдать на небе
- Спланировать наблюдательную сессию
- Выбрать объекты для наблюдения
- Узнать лучшее время для наблюдения

## Протокол действий

### Шаг 1. Уточни цель наблюдения
Определи, что именно пользователь хочет увидеть:
- Конкретный объект (планету, звезду, созвездие)?
- Тип объектов (все планеты, яркие звёзды)?
- Или он хочет рекомендацию — «что интересного посмотреть»?

### Шаг 2. Собери данные об объектах
Используй доступные инструменты:
- `list_planets` / `get_planet` — для планет
- `list_stars` / `get_star` — для звёзд
- `get_constellation` — для созвездий и связанных объектов
- `search_objects` — если цель размытая

### Шаг 3. Оцени наблюдаемость
Для каждого объекта оцени:
- **Видимая звёздная величина** (apparent_magnitude) — чем меньше число, тем ярче. Объекты ярче +6 видны невооружённым глазом.
- **Сезон наблюдения** (best_season у созвездий) — подходит ли текущий период?
- **Расстояние** — близкие планеты (< 5 AU) обычно яркие и заметные.

### Шаг 4. Составь план наблюдения
Сформируй рекомендацию в виде структурированного плана:

```
📋 ПЛАН НАБЛЮДЕНИЙ
═══════════════════
🎯 Цель: [что наблюдаем]
🕐 Лучшее время: [сезон / часы]
🔭 Рекомендуемое оборудование: [невооружённый глаз / бинокль / телескоп]

Объекты для наблюдения:
1. [Объект] — [почему интересен] — [как найти на небе]
2. ...

💡 Советы:
- [практический совет по наблюдению]
```

### Шаг 5. Добавь контекст
- Упомяни связанные объекты (например, если наблюдаем Орион — упомяни туманность M42)
- Дай интересный факт о каждом объекте (fun_fact)
- Если объект не виден в текущий сезон — предложи альтернативу

## Стиль ответа
- Будь энтузиастом, вдохновляй на наблюдения
- Используй эмодзи для визуальной структуры (🌟, 🔭, 🪐)
- Всегда объясняй, КАК найти объект на небе (ориентиры, направления)
- Давай практические советы (уйти от городского света, дать глазам привыкнуть к темноте)
